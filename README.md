# Sqily

Plateforme de communication scolaire et de validation mutuelle de compétences, proposée par la Haute Ecole Pédagogique Vaud.
C'est une application web écrite avec Ruby on Rails.

## Développement

Lancer `docker compose up --build` pour démarrer le base de données et le serveur.

## Déploiement

Chaque modification sur `master` est automatiquement déployée en production.
